# GNR Notifiable Diseases Dashboard

R Markdown source for the GNR Public Health Notifiable Diseases Dashboard. The dashboard tracks vaccine-preventable, vectorborne, and enteric illnesses across the Gwinnett-Newton-Rockdale health district.

Pushes to `main` automatically compile the dashboard via GitHub Actions and publish it to GitHub Pages. The WordPress site embeds that URL via iframe.

## Update workflow (for the GNR team)

1. Open the project folder on your computer and update data files (`data/clean/`) or R code.
2. Open **GitHub Desktop**. It will show what changed.
3. Type a short description (e.g., "Updated Q1 2026 data") and click **Commit to main**.
4. Click **Push origin**.
5. Wait a few minutes. The dashboard on the website will refresh automatically.

You do **not** need R installed on your computer. GitHub builds the dashboard for you on every push.

If a push doesn't update the site within ~10 minutes, check the **Actions** tab on GitHub for error details, or contact Applied Imagination.

## Repository layout

```
GNRND_Dashboard.Rmd      Main R Markdown dashboard
R/                       Pipeline scripts (data load, plots, demographic pies)
  00_Report Generation Pipeline.R   Orchestrator — sourced by CI
narratives/              R Markdown chunks for each disease program
data/clean/              Deidentified input data (driven by combo_deidentified_clean.qs)
data/                    Lookup tables (county populations, disease program mapping)
output/                  Pre-rendered .rds plot objects (generated by CI, gitignored)
reports/                 Alternate / draft report outputs
OGheader.html            Header includes (fonts, container padding, loader)
header.html              Accessibility enhancements (skip link, focus styles)
renv.lock                R package versions — restored by CI on every build
```

## Building locally (optional)

Requires R 4.3+. From the repo root, in R:

```r
renv::restore()
source("R/00_Report Generation Pipeline.R")
```

Output lands at `final_dashboard.html`.
